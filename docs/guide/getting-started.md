# 快速开始

本文档站基于 [VitePress](https://vitepress.dev/) 构建，使用 Markdown 编写所有文档。

## 如何使用

### 浏览文档

- **左侧边栏**：展开文档目录，快速跳转
- **顶部搜索栏**：输入关键词搜索（快捷键 `Ctrl+K` / `Cmd+K`）
- **页面导航**：右侧展示当前页面的标题大纲
- **上一页/下一页**：页面底部可连续浏览

### 搜索技巧

1. 按 `/` 或 `Ctrl+K` 打开搜索框
2. 输入关键词，支持模糊搜索
3. 用 ↑ ↓ 方向键切换结果，Enter 跳转

### 主题切换

点击顶部导航栏右侧的图标切换浅色/深色模式。

## 如何贡献

1. 在 `docs/` 目录下找到对应板块
2. 新建或编辑 `.md` 文件
3. 更新侧边栏配置（`docs/.vitepress/config.ts`）
4. 本地预览确认后提交

## 本地运行

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build

# 预览构建产物
npm run preview
```
